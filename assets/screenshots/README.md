# Screenshots

Place your workflow screenshot(s) in this folder. Recommended file names
(these are referenced in the main README and also help with image SEO,
since Google indexes image file names and alt text):

- `workflow-overview.png` — full n8n canvas, all 14 nodes visible
- `ai-brain-1-response-engine.png` — AI Brain 1 node configuration (optional)
- `ai-brain-2-conversation-analyst.png` — AI Brain 2 / decision JSON output (optional)
- `admin-notification-example.png` — example admin alert on Messenger (optional, blur any personal info)

Tips before uploading:
1. Blur or crop out any API keys, access tokens, phone numbers, or personal profile IDs visible in the screenshot.
2. Export at a readable resolution (at least 1600px wide) so text in the n8n canvas is legible.
3. Keep the file name descriptive and hyphenated — avoid generic names like `Screenshot1.png`.
